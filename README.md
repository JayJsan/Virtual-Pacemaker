# FPGA - Pacemaker

## Switches

> ### **SW0**
>
> #### - 0 = UART MODE
>
> #### - 1 = BUTTONS MODE

> ### **SW1**
>
> #### - 0 = C IMPLEMENTATION MODE
>
> #### - 1 = SCCHARTS IMPLEMENTATION MODE

## Buttons

> ### **KEY0**
>
> #### - PRESS = PACE ATIRAL

> ### **KEY1**
>
> #### - PRESS = PACE VENTRICLE

> ### **KEY0**
>
> #### - PRESS = RESTART MAIN LOOP
